# 1.该微服务<br>
1.1、提供针对mysql导出markdown的数据表结构<br>
1.2、提供针对clickhouse导出markdown的数据表结构<br>
1.3、提供针对elasticsearch导出markdown的数据表结构<br>
1.4、提供针对mongodb导出markdown的数据表结构<br>


# 2.启动
```shell 
打包编译后,启动
java -jar java-mysql-markdown-1.0.0.jar
```
